# 代码注释的重要性

一份好的代码，不仅要求代码的执行效率高，其实读取作者的逻辑效率高也是一个很重要的体现。

那么如何在代码中体现作者的逻辑呢？一个很重要的辅助就是注释了。

# 特殊的注释

基本的注释，我这里就不再多说了，因为每一种语言对应的注释都会有相应的不同，今天我想提到的是一些比较特殊的注释。

**TODO**

```c
// TODO 标识处有功能代码待编写
```

**FIXME**

```c
// FIXME 代码需要修正，甚至代码是错误的，不能工作，需要修复
```

**XXX**

```c
// XXX 代码虽然实现了功能，但是实现的方法有待商榷，希望将来能改进
```
